Cassaforte

A young Clojure client for Apache Cassandra 1.2+

Get Started! » Read doc guides » Join the mailing list » Contribute »

Feature complete

Connect to Cassandra cluster, manipulate wide rows via thrift interface, use latest CQL features via new shiny native CQL client interface.

Battle-tested

Built out of necessity to have a complete, stable, easy-to-use and understand client, that doesn't hide internals and allows to operate in high and low-level abstractions.

Cassandra 1.2+ features

Utilize features like Lists, Sets, Maps, composite keys, indexing, range queries.

Modern Cassandra Features

Utilize features like lists, sets, maps, composite keys, indexing, and range queries.

CQL 3

Party like it's 3.0: awesome CQL features, waiting for you to use them.
