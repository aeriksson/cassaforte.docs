Cassaforte

A young idiomatic Clojure client for Apache Cassandra 1.2+.

Get Started! » Read doc guides » Join the mailing list » Contribute »

Complete

Thrift, native CQL3 and clustered CQL 3 clients to choose from. No compromises.

Clojuric

Work with Clojure data structures. No unnecessary abstractions. Simple and easy.

CQL 3

Party like it's 3.0: awesome CQL features, waiting for you to use them.

The API You Won't Hate

The power of Cassandra at your fingertips, with an API that does not make you want to kill kittens.

Batteries Included

Additional bells and whistles to make your life easier, within reason.

Modern Cassandra Features

Utilize features like lists, sets, maps, composite keys, indexing, and range queries.
